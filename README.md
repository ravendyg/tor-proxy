Multiple tor instances manager.

&nbsp;

## Dependecies
node 6.10.2, tor

&nbsp;

## Installation
```
npm i
```

&nbsp;

## Setup
Requires two ports for each tor instance.

To override default config create ./lib/config.json file and put new values there.

Provide API_KEY in config.json

&nbsp;

## Start
```
npm start
```
